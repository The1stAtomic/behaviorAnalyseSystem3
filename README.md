# behaviorAnalyseSystem

# Prerequisites
- You need Python 3.11 installed

# Download the project
+ Option 1 - Clone with git:
  git clone https://github.com/your-username/your-repo.git
  - Change directory into the project:
    - Window:
      cd your-repo
    - Mac:
      cd your-repo
+ Option B - Download ZIP:
  - Download: Click “Code” > “Download ZIP” on the repository page.
  - Extract: Unzip it.
  - Change directory into the folder:
    - Windows:
      cd path\to\unzipped\folder
    - Mac:
      cd /path/to/unzipped/folder

# Create and activate a virtual environment
+ Create the environment:
  - Windows:
    python -m venv venv
  - Mac:
    python3 -m venv venv
+ Activate the environment:
  - Windows (Command Prompt):
    venv\Scripts\activate
  - Windows (PowerShell):
    .\venv\Scripts\Activate.ps1
  - Mac:
    source venv/bin/activate
Tip: After activation, your terminal prompt will show “(venv)” at the start. That’s how you know it worked.

# Install dependencies
+ Using requirements.txt:
  - Windows:
    pip install -r requirements.txt
  - Mac:
    pip3 install -r requirements.txt

# Lastly, run main.py
